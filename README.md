FortressAuth is a secure desktop authentication system built in Python that simulates how real-world applications handle users, credentials, and access control.

This project focuses on building real security logic instead of toy examples.

## Features
- User registration and login
- Secure password hashing (no plain-text storage)
- Failed login attempt tracking
- Automatic account lock after multiple failures
- Login activity logging
- Modular Python architecture
- Modern Tkinter-based GUI

## Tech Stack
- Python 3.12+
- Tkinter
- JSON-based user storage
- hashlib for password security
